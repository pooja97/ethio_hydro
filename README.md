# ETHIO_HYDRO

It is an interactive dashboard for time series temperature and precipitation data analysis created using Python, Pandas, and Pandarallel
(for parallel processing), geopy, Streamlit, Plotly, and Altair charts.  

This dashboard is for Ethiopia's rain and temperature data collected from the SWAT database (`https://swat.tamu.edu/data/`).  
The dashboard uses time-series data from 2001 to 2019 of Ethiopia's every latitude and longitude.


## To Run

Go to the project folder and install the required packages.

Packages:

pandas: pip3 install pandas 
streamlit: pip3 install streamlit
geopy: pip3 install geopy
Pandarallel: pip3 install pandarallel
plotly: pip3 install plotly
Altair charts: pip3 install altair

Command to run: `streamlit run app.py`
